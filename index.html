Ingeniería de software

Es la rama de la ingeniería que estudia todo lo relacionado con la informática o sistemas de computación, con una orientación metódica, ordenada y cuantificable al incremento, ejecución y conservación del software.

Conceptos fundamentales

  • Definición de un modelo

    Un modelo es una representación de un objeto, en este caso es la representación de una seria de etapas (ciclo de vida) en la cual se realizan las fases para obtener un resultado.

  • SCRUM

    La metodología SCRUM, se basa en iteraciones incrementales en la cual tienen entregas parciales, así como reunión con los clientes, el cual aporta valor al desarrollo del software.

  • Requerimientos

    Es una condición, capacidad o funcionalidad que debe estar presente en un producto o servicio, para satisfacer las necesidades solicitadas.

  • Definición de requisitos

    Para considerar una característica o función un requisito es necesario considerar los siguientes puntos:
    • 1.-Condición o función que un usuario necesita para resolver un problema.
    • 2.-Condición o capacidad que un sistema requiere para resolver una problemática.

  • "Arquitectura de software"

    Tubos y filtros

    Esta arquitectura consiste en dividir la información obtenida por tubos y filtros, en la cual cada filtro procesa la información y se lo manda por medio de un tubo al siguiente filtro u otra etapa.

API

Una API (Application Programming Interface) es un conjunto de definiciones y protocolos, utilizados para desarrollar e integrar software de aplicaciones.
